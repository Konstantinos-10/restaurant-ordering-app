# Mobile Restaurant Ordering App

**Design Direction**:[Figma File](https://www.figma.com/design/F2nDEYLnYvdlW6n9X3K4iw/Mobile-Restaurant-Menu--Copy-?node-id=0-1&p=f&t=J0fiKKWJBwHhXC1T-0)

## 4 Stages of The app

### 1. Default State

Rendered using javascript.

When user clicks add button they take you to the next state.

### 2. Pre-checkout State

Here we have a new section "Your Order"

Once clicked to the complete button we go to the next state

### 3. Checkout Payment Modal State

Payment Modal.

HTML Form with 3 fields and a button

If a user clicks to pay with missing fields, the user should get a prompt by the browser.

### 4. Order-Complete State

Once the form is completed, the order section shows a success message

## Requirements

- Follow the design spec
- Render the menu options using Javascript
- Be able to add/remove items
- Have a payment modal with compulsory form inputs

### Stretch goals

- Change the theme (whatever you want)
- Add a "meal deal" discount
- Allow users to rate their experience (5 stars)
